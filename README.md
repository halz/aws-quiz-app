# AWS Quiz App

AWS Certification Exam Practice Quiz Application built with Next.js 14.

## Features
- 1,183 practice questions across 4 AWS certifications
- - CLF-C02 (Cloud Practitioner): 245 questions
  - - AIF-C01 (AI Practitioner): 253 questions
    - - SAA-C03 (Solutions Architect Associate): 465 questions
      - - SOA-C02 (SysOps Administrator Associate): 220 questions
       
        - ## Tech Stack
        - - Next.js 14 (App Router)
          - - TypeScript
            - - Tailwind CSS
